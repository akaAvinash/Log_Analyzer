#### Log Analyzer ####

The Log Analyzer is a Python-based tool designed to simplify log file analysis. It allows users to extract valuable insights from log files, identify patterns, and troubleshoot issues effectively.

Prerequisites:
Before using the Log Analyzer, ensure you have Python installed on your system. To check your Python version, open a terminal or command prompt and enter the following command:

      $ python --version

If Python is not installed or you have an outdated version, visit the official Python website (https://www.python.org) to download and install the latest version compatible with your operating system.

Getting Started

To run the Log Analyzer, follow these steps:

Clone this repository or download the source code as a ZIP file.
Open a terminal or command prompt and navigate to the directory containing the downloaded files.
Run the Log Analyzer using the following command:
python log_analyzer.py  (Include version of Python installed, E.g. python3.8 log_analyzer.py)
The tool will prompt you to enter the search strings for log lines you wish to extract. Provide the desired strings and press Enter.
The Log Analyzer will process the log files and display the extracted log lines that match the provided search strings.
